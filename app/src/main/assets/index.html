使い方

  1. PCで「スマホでアソビライト連携」のQRコードを表示してください。
  2. 本画面の右上にある をタップしてください。カメラに切り替わりますので、QRコードを撮影してください。
  3. 続けて、本端末にアソビライト画面が表示されますので「接続」をタップしてください。
  4. アソビライトが表示された状態で、本端末を振ると、アソビライトを振ることができます。

ヒント

  • PC側は映像のみのフルスクリーンにすることも可能です。詳細は README の Q&A をご覧ください。
  • 振っている間に誤タッチで別画面に切り替えてしまうときは、本画面の右上の端にある を押して全画面表示をお試しください。
